# List Accounts

- An authenticated user can view the accounts in the selected entity.
  - The list is alphabetized.
  - The list shows the fully qualified names of the accounts. The fully
    qualified name is the account name prefixed by the account parent
    names. E.g., if there is an account "Car" that has a parent account
    named "Savings", then the fully qualified name of "Car" is "Savings/Car".
  - The list is grouped by account type and the types are listed in this order:
    - Asset
    - Liability
    - Equity
    - Income
    - Expense
- An authenticated user can view the accounts in a specified entity.
- The level of nesting can be limited with an option.
